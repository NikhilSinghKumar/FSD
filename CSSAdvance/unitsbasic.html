Units in CSS

px, em: element, rem: root element

%, vw: viewport width, vh: viewport height

Absolute units: px (just fixed). not dependent on any other element

Relative units (provides some kind of responsiveness): em, rem, %, vw, vh

  • px: consistent size irrespective of screen size;
  • em: Relative to its parent element;
  • rem: Relative to the root element (html tag);
  • vw, vh: relative to Viewport (available space on screen);
  • %: relative to its parent element;
Most Browsers, default font-size: 16px
  • html: 16px;
  • 1em: 16px;
  • 1rem: 16px;
  • 100%: 16px;
